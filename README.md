# Mastotuner

Like [GanerCodes/AutotuneBot](https://github.com/GanerCodes/AutotuneBot) but for Mastodon.

## Installation

### Dependencies
```
$ pip install bs4 mastodon.py requests yt-dlp
```

You'll need `wine` if you're on Linux.
